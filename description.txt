[h1]Timetables (1.3)[/h1]
I can proudly present an update to TF2s most popular mod: Timetables!
This mod adds timetables to the game.
Timetables can be configured for each line and each station individually. There are three types of timetable constraints that can be added to a station:
[list]
[*] [b] Arrival/Departure[/b]: You can add arrival/departure timeslots. The train chooses the closest arrival time slot when entering the station, and then wait until the departure time. 
[*] [b] Unbunch[/b]: You can set the frequency and the mod will enforce that the trains will depart this station with the given frequency. 
[*] [b] Auto Unbunch[/b]: You can set the line to automatically unbunch each vehicle by the frequency of the line.
[/list]
Trains will depart right away when they are late without interference from the mod. The timetable mod uses the game the stop command and can therefore only enforce longer waiting times. It won't make trains drive faster or load/unload quicker.

Timetables are only working when the checkbox on the line overview is ticked.

[h2]Updates from the original mod[/h2]
You can view the original mod [url=https://steamcommunity.com/sharedfiles/filedetails/?id=2408373260]here[/url]
This mod is a fork of the original and adds the following features:
[list]
[*] Supports multiple vehicles at the same station
[*] Remembers the timetable slot of vehicles after a load-save cycle
[*] Adds Auto Unbunch to automatically unbunch vehicles regardless of how many are on the line
[*] Updated colours to support the Spring Update line colours
[*] Minimum waiting time is supported
[*] Maximum waiting time is supported
[/list]

[h2]Things to beware of[/h2]
[list]
[*] Saves can be transferred from the original mod to this mod
[*] Saves [b]cannot[/b] be transferred from this mod back to the original mod
[*] Therefore, I would recommend making backups of saves before migrating to this mod
[*] The behaviour for picking timetable slots has changed. Vehicles now pick the nearest slot, regardless of whether it is in the past or the future.
[*] To support multiple vehicles at the same station, vehicles will pick the next consecutive available arrival time when the nearest is taken.
[/list]

[h1]Fahrpläne (1.3)[/h1]
Ich kann stolz ein Update für TF2s beliebteste Mod präsentieren: Fahrpläne!
Dieser Mod fügt dem Spiel Fahrpläne hinzu
Die Fahrpläne können individuell für jede Linie und Station angepasst werden. Es gibt 3 Typen von Fahrplan Bedingungen die einer Station hinzugefügt werden können:
[list]
[*] [b] Arrival/Departure[/b]: Es ist möglich Ankunft/Abfahrt Slots der Station hinzuzufügen. Der ankommende Zug entscheidend anhand der Ankunftszeit welchen Slot er nimmt und wartet dann bis zur Abfahrtszeit in der Station.
[*] [b] Unbunch[/b]: Es kann eine Frequenz gesetzt werden. Der Zug fährt in der gesetzten Frequenz am Bahnhof ab.
[*] [b] Auto Unbunch[/b]: Sie können die Linie so einstellen, dass jedes Fahrzeug anhand der Frequenz der Linie automatisch entbündelt wird.
[/list]
Bei Verspätung fahren die Züge nach dem ein und ausladen direkt weiter. Der Mod verwendet das Stop Kommando aus dem Spiel und kann deshalb nur Züge länger warten lassen, jedoch keine Züge schneller fahren lassen oder die Ladezeitn beschleunigen!

Damit die Fahrpläne funktionieren muss die kleine Checkbox auf der Linienübersicht aktiviert sein!

[h2]Updates from the original mod[/h2]
Sie können den Original-Mod [url=https://steamcommunity.com/sharedfiles/filedetails/?id=2408373260]hier ansehen[/url]
Dieser Mod ist ein Fork des Originals und fügt die folgenden Funktionen hinzu:
[list]
[*] Unterstützt mehrere Fahrzeuge an derselben Station
[*] Merkt sich den Fahrplan-Slot von Fahrzeugen nach einem Ladesparzyklus
[*] Fügt Auto Unbunch hinzu, um Fahrzeuge automatisch zu entbündeln, unabhängig davon, wie viele sich auf der Linie befinden
[*] Aktualisierte Farben zur Unterstützung der neuen Farben aus dem Frühlings-Update
[*] Mindestwartezeit wird unterstützt
[*] Maximale Wartezeit wird unterstützt
[/list]

[h2]Wichtige Änderungen[/h2]
[list]
[*] Saves können vom Original-Mod auf diesen Mod übertragen werden
[*] Speicherstände können [b]nicht[/b] von diesem Mod zurück auf den Original-Mod übertragen werden
[*] Daher würde ich empfehlen, Sicherungskopien von Spielständen zu erstellen, bevor Sie zu diesem Mod migrieren
[*] Das Verhalten bei der Auswahl von Stundenplan-Slots hat sich geändert. Fahrzeuge wählen jetzt den nächstgelegenen Stellplatz, unabhängig davon, ob er in der Vergangenheit oder in der Zukunft liegt.
[*] Um mehrere Fahrzeuge an derselben Station zu unterstützen, wählen die Fahrzeuge die nächstfolgende verfügbare Ankunftszeit aus, wenn die nächste genommen wird.
[/list]
